Kebijakan Privasi

Terakhir diperbarui: 16 Juni 2026

Di Trackora Financial (karya Zulfinfinity Digital Labs), privasi Anda adalah prioritas utama kami. Kebijakan ini menjelaskan bagaimana kami menangani informasi Anda saat Anda mengunjungi website kami atau menggunakan Web App Trackora.

1. Data Keuangan Anda 100% Milik Anda

Trackora memiliki arsitektur yang unik. Tidak seperti aplikasi keuangan pada umumnya, kami tidak memiliki server database. Semua data keuangan, riwayat transaksi, saldo, dan pengaturan aplikasi Anda disimpan secara eksklusif di dalam akun Google Drive / Google Sheets Anda sendiri.

Kami (Zulfinfinity Digital Labs) tidak memiliki akses, tidak dapat melihat, tidak menyimpan, dan tidak menyebarkan satu byte pun data finansial Anda.

2. Data yang Kami Kumpulkan

Kami hanya mengumpulkan informasi minimum yang diperlukan untuk proses transaksi dan pelayanan pelanggan:

  • Informasi Pembelian: Saat Anda membeli Trackora, pembayaran diproses secara aman oleh pihak ketiga (payment gateway Mayar.id). Kami hanya menerima informasi profil dasar (Nama, Email, dan Nomor WhatsApp) untuk mengirimkan kode lisensi dan memberikan dukungan teknis (support).
  • Data Analitik Web (Opsional): Kami mungkin menggunakan alat analitik standar di landing page kami untuk memahami bagaimana pengunjung menggunakan situs web kami. Ini tidak terhubung dengan data di dalam Web App Anda.

3. Keamanan Data

Karena data Anda disimpan di Google Drive Anda, tingkat keamanan data Anda bergantung pada keamanan akun Google Anda. Kami sangat menyarankan Anda untuk mengaktifkan Autentikasi Dua Langkah (2FA) pada akun Google Anda.

4. Perubahan pada Kebijakan Privasi

Kami dapat memperbarui kebijakan ini dari waktu ke waktu. Perubahan akan langsung efektif setelah kami perbarui di halaman ini.
